## Sobre o Projeto
Neste projeto irá conter os exercícios na qual solucionei para práticar o que eu aprendi nos meus estudos:
- [x] Estrutura de repetição.
- [ ] Estrutura de dados.
- [ ] Hash Map.
- [ ] Hash Set.
- [ ] POO.
- [ ] Encapsulamento.
- [ ] Herança. Polimorfismo.
- [ ] Exceções.
- [ ] Construtor.
- [ ] Wrappers.

## Requisitos
 * IDE de sua escolha(VScode / Eclipse).
 * Java 17
 * Docker
